# EAP Documentation

Welcome to the **Enterprise Application Project (EAP)** documentation wiki.

## What is EAP?

The Enterprise Application Project is a trainee program that prepares participants for careers in Dutch corporate IT through hands-on Scrum-based software development. Teams of trainees build a real web application from scratch over multiple sprints — with staff acting as coaches, not managers.
